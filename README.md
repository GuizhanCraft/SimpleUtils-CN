# 简易工具(SimpleUtils)

一个小型粘液科技(Slimefun)扩展，添加了有用的东西。



## 特性

- 融合工作台(Enhanced Workbench): 在同一个菜单里合成原版与粘液科技物品

- 简易电梯 - 跳跃上楼，蹲下下楼

- 扳手 - 快速破坏货运节点，电容，以及电动机器
